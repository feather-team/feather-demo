feather会持续更新一些常见有用的插件

  • feather-postprocessor-inline-compress(内联样式以及script压缩): 
    
<script>
(function(window, undefined){
    console.log(window, undefined);
})(window);
</script>
 
<!--压缩这段代码-->
<script feather-inline-compress>
(function(window, undefined){
    console.log(window, undefined);
})(window);
</script>
 
<style type="text/css" feather-inline-compress>
*{
    font-size: 12px;
}
</style>
  • less集成：feather中内置了less插件，任何css结尾的文件都可以使用less语法。 
    a{
    &:hover{
        color: red;
    }
     
    span{
        display: inline-block;
    }
}
  • livereload功能内置：feather release -L即可
