feather中的sprite功能会对合并的css自动启用

feather.config.set('pack', {
	'static/combine.css': 'static/**.css'
});

/*1.css*/
div{
	background: url(/static/1.png?__sprite);
}

a{
	background: url(/static/2.png?__sprite);
}

/*2.css*/
i{
	background: url(/static/3.png?__sprite);
}

如果有其他特殊，或者更详细的具体需求，可见fis-spriter-csssprites
