feather-ui组件库是一套基于jquery/zepto的简单，轻量，且遵循暴露核心API，易扩展，且代码可读性高思想的前端ui组件库。

feather-ui只提供了一些网站上的常用组件以及一些简单的特效，因使用者的需求往往是很简单的，可能仅仅只是需要一些简单的特效。另外一方面，一般开源的组件库往往会实现各种功能以便组件库看上去更完善，然而对于需要定制的使用者其实是一种灾难，往往因为臃肿的代码而不敢下手。所以feather-ui提倡暴露更多的核心api，让使用者更加容易的去进行扩展，以便适应自己的业务场景。注：feather-ui不会更新一般比如图表类，表格类等插件，因这些插件已经真的够多的了！

注：feather-ui兼容了模块化加载，以及传统加载方式，如果是非模块下的调用，则需要提前知道组件的依赖是哪些，具体的可以进组件文件中查看，比如mask组件，依赖于jquery.js，则需要提前加载jquery，方式如下:

<link rel="stylesheet" href="/static/mask.css" />
<script src="/static/jquery.js"></script>
<script src="/static/mask.js"></script>
<script>
//调用时
var mask = new FeatherUi.Mask({});
mask.open();
</script>
